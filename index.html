HTML — 구조 만들기

HTML은 웹페이지의 뼈대(구조)를 만드는 언어예요. 태그로 제목·문단·이미지·버튼 등을 표현합니다.

기본 구조

<!DOCTYPE html>
<html>
  <head> ... 문서 설정 ... </head>
  <body> ... 화면에 보이는 내용 ... </body>
</html>

코딩 캔버스에서는 <body> 안의 내용만 HTML 탭에 적으면 돼요. 머리말/스타일/스크립트는 자동으로 합쳐집니다.

자주 쓰는 태그

<h1>~<h6>제목 (숫자가 작을수록 큼)
<p>문단(텍스트)
<a href="...">다른 페이지로 가는 링크
<img src="..." alt="...">이미지 (alt은 설명)
<ul> <ol> <li>목록 (순서 없음/있음)
<button>클릭 버튼
<input>입력창
<div> <span>내용을 묶는 영역/구간

속성은 태그에 정보를 더해요. 예) <a href="주소">, <img src="그림" alt="설명">

CSS — 꾸미기

CSS는 HTML을 꾸미는(디자인) 언어예요. 색·글꼴·여백·배치를 정합니다.

작성 방법

선택자 {
  속성: 값;
}

/* 예시 */
p { color: blue; font-size: 16px; }

선택자

p모든 <p> 태그
.boxclass="box" 인 요소
#titleid="title" 인 요소

자주 쓰는 속성

color / background글자색 / 배경색
font-size글자 크기
padding / margin안쪽 / 바깥쪽 여백
border / border-radius테두리 / 모서리 둥글기
display: flex가로 정렬(Flexbox)

박스 모델: 내용(content) → 안쪽 여백(padding) → 테두리(border) → 바깥 여백(margin) 순서로 감싸져요.

JavaScript — 움직이게 하기

JS는 웹페이지를 움직이게(상호작용) 만드는 언어예요. 클릭·입력 등에 반응합니다.

변수와 자료형

let name = "홍길동";   // 문자열
const age = 20;        // 숫자
let isOn = true;       // 참/거짓 (boolean)
let list = [1, 2, 3];  // 배열

함수

function hello(){
  console.log("안녕!");
}
hello();

화면 다루기 (DOM)

const el = document.querySelector("#box");
el.textContent = "바뀐 내용";
el.style.color = "red";

이벤트 (클릭 등에 반응)

button.addEventListener("click", () => {
  alert("클릭됨!");
});

console.log(...)의 결과는 오른쪽 아래 콘솔 패널에서 볼 수 있어요. 오류도 콘솔에 빨간색으로 표시됩니다.

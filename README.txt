# 코딩 캔버스(Coding Canvas)

## 소개
코딩 캔버스는 좌측에 코드 입력 창, 우측에 결과 미리보기 창이 있는 웹 기반 코딩 실습 도구입니다.
아이콘 버튼을 이용하여 실행, 자동실행 전환, 라이트/다크 모드 전환, 업로드/다운로드, 초기화 기능을 제공합니다.

## 주요 기능
1. **코드 실행** : 좌측 코드 내용을 우측 결과 창에 즉시 반영합니다.
2. **자동실행** : 코드 수정 시 실시간으로 결과 창이 갱신됩니다.
3. **라이트/다크 모드 전환** : UI 테마를 변경합니다.
4. **파일 업로드** : JSON 또는 TXT 형식의 저장된 프로젝트를 불러옵니다.
5. **파일 다운로드** : 현재 프로젝트를 JSON 또는 TXT 형식으로 저장합니다.
6. **로고 클릭 초기화** : 좌측 상단 로고 클릭 시 초기 화면으로 되돌립니다.
7. **모바일 최적화** : 데스크톱과 모바일 환경 모두에서 사용 가능합니다.

## UI 특징
- 좌측: 코드 입력 탭 (HTML, CSS, JS)
- 우측: 결과 미리보기 창
- 하단: 푸터(제작자 정보) 화면 하단 고정
- 로고 크기: 기존 대비 2배 확대

# 코딩 캔버스 (Coding Canvas)

## 소개
코딩 캔버스는 브라우저에서 HTML·CSS·JS를 바로 작성하고, 결과 미리보기와 콘솔 출력을
실시간으로 확인할 수 있는 코드 테스트 학습 플랫폼입니다. 별도 설치 없이 index.html
하나만 열면 동작합니다.

## 주요 기능
1. **실시간 미리보기** : 좌측 코드(HTML/CSS/JS)를 우측 결과 창에 즉시 반영합니다.
2. **콘솔 패널** : iframe 내부의 console.log/info/warn/error 출력과 실행 중 발생한
   오류·Promise 거부를 우측 하단 콘솔 패널에서 색상별로 확인할 수 있습니다.
   콘솔이 닫혀 있을 때 오류가 나면 콘솔 아이콘에 빨간 점으로 알려줍니다.
3. **학습 도움말** : HTML/CSS/JS 기초 가이드, 단축키 안내, 그리고 바로 불러올 수 있는
   예제를 난이도·주제별 5개 카테고리(12개)로 제공합니다.
   - ① 기초 — 첫걸음 : 기본 인사, 배경색 바꾸기
   - ② 레이아웃 & 디자인 : 프로필 카드, 사진 갤러리, 버튼 스타일 모음
   - ③ 인터랙션 & 이벤트 : 클릭 카운터, 할 일 목록, 탭 메뉴, FAQ 아코디언
   - ④ 애니메이션 & 그래픽 : 로딩 스피너, 디지털 시계
   - ⑤ 미니 게임 & 알고리즘 : 가위바위보, 숫자 맞히기
4. **예제 코드 설명 띠** : 예제를 불러오면 현재 탭(HTML/CSS/JS)이 담당하는 역할을
   에디터 상단에 안내해 줍니다.
5. **자동실행** : 코드 수정 시 실시간으로 결과가 갱신됩니다(켜짐/꺼짐 토글).
6. **상태 표시줄** : 현재 언어, 커서 위치(줄/열), 줄 수, 글자 수를 표시합니다.
7. **라이트/다크 모드 전환**, **데스크톱/모바일(375px) 미리보기 전환**.
8. **새 탭에서 열기** : 결과를 새 탭에서 독립적으로 확인합니다.
9. **코드 복사** : 현재 탭의 코드를 클립보드로 복사합니다.
10. **파일 업로드/다운로드** : 다운로드는 JSON(프로젝트 전체)·HTML(단일 실행 파일)·TXT
    형식을 지원하며, 불러오기는 JSON·HTML·TXT·CSS·JS 파일을 지원합니다.
11. **자동 저장** : 작성한 코드·테마·레이아웃이 브라우저에 저장되어 다음에 이어서 작업할 수 있습니다.
12. **모바일 최적화** : 데스크톱은 좌우 분할, 모바일은 세로 스택으로 자동 전환됩니다.

## 단축키
- 실행 : Ctrl/⌘ + Enter
- 들여쓰기 : Tab (공백 2칸)
- 패널 폭 조절 : 가운데 구분선 드래그 / 더블클릭 시 균등 분할

## UI 구성
- 좌측 : 상단 아이콘바 → 코드 탭(HTML/CSS/JS) → 예제 설명 띠 → 에디터(줄 번호) → 상태 표시줄
- 우측 : 미리보기 도구바 → 결과 미리보기 → 콘솔 패널
- 하단 : 푸터(제작자 정보) 고정
- 좌측 상단 로고 클릭 시 전체 초기화(확인 후 진행)
